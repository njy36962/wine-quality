## Description

This interactive web application allows users to predict the quality of white wine based on its chemical properties using a pre-trained machine learning model.

Features include:
* Adjustable sliders for 11 key wine features (acidity, sugar, alcohol, pH, etc.)
* Real-time wine quality prediction
* Option to save predictions for future reference

Interactive visualizations:
* Feature importance chart
* Wine quality distribution
* Scatter plots of features vs. quality
